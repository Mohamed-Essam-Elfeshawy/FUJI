## 🚀 خطوات التشغيل السريع

### 1. التشغيل المحلي
```bash
# فتح مجلد المشروع
cd fuji-fd-website

# تشغيل خادم محلي (اختر أحد الطرق)
python -m http.server 8000
# أو
npx serve .
# أو
php -S localhost:8000

# فتح المتصفح على
http://localhost:8000
```

### 2. التطوير مع Tailwind
```bash
# تثبيت المتطلبات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء للإنتاج
npm run build
```
